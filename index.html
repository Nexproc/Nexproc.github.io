Nexproc
Let's Kick this Pig

Nexproc is a lightweight, single-page team productivity app inspired by Asana. It is deployed on Rails' RESTful MVC framework and utilizes Backbone's models and collections to represent the data that is persisted on its own SQL database. Nexproc's primary functionality and overall DRYness comes from the use of a custom API that is patched over Backbone's existing data structures.
